# CellPacker

FreeCAD battery pack layout generator for cylindrical cells.

## Features

- Hexagonal packing
- Shape constrained packing
- S/P group assignment
- Busbar generation
- Edge-aligned layouts
- 2D and 3D visualization

## Installation

Copy BatteryPackLayoutTool_v2.FCMacro to:

Windows:
%APPDATA%\FreeCAD\Macro

Linux:
~/.local/share/FreeCAD/Macro
